## 核心观点速览

### 生附子 vs 炮附子 vs 生硫磺
生附子 ──→ 阴寒（病在里、阳气极虚、四逆证：脚冷至膝 / 手冷至肘）
炮附子 ──→ 表虚（表阳不固、汗出恶风、卫气不固）
生硫磺 ──→ 水中火（命门火衰、水肿、关节肿痛、阴寒水肿）

### 六经传变规律

太阳（表）→ 阳明（里热）→ 少阳（半表半里）
↓ 失治误治
太阴（脾寒）→ 少阴（心肾阳虚）→ 厥阴（阴阳逆乱 / 上热下寒）

### 五味饮食观
| 厚味 | 伤害 | 淡味 | 归属 |
|---|---|---|---|
| 咖啡（浓酸） | 伤脾→伤肾→骨伤 | 蔬菜淡苦 | 入心 |
| 牛奶（酸性） | 伤脾/胰 | 天然蔗糖 | 缓急止痛 |
| 重盐 | 伤骨/心 | 天然海盐 | 适量护正 |
