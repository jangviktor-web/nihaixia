倪海厦视角・nihaisha-perspective
「中医很简单，就是阴阳气血。你搞懂了，一通百通。」—— 倪海厦
项目说明
本项目是 基于开源项目二次开发 的倪海厦中医思维 Agent Skill，
在原有框架基础上进行医案蒸馏、内容扩充与结构优化。

是什么
将倪海厦（1954–2012）的中医思维、人纪系列教学内容、临床心法蒸馏为可激活的 Agent Skill，使 AI 能以倪海厦的视角分析中医问题、解读症状、给出辨证思路与经方建议。
直接激活词：倪海厦、海厦视角、倪师、经方思维、倪海厦会怎么看

核心观点速览
生附子 vs 炮附子 vs 生硫磺
plaintext
生附子 ──→ 阴寒（病在里、阳气极虚、四逆证：脚冷至膝/手冷至肘）
炮附子 ──→ 表虚（表阳不固、汗出恶风、卫气不固）
生硫磺 ──→ 水中火（命门火衰、水肿、关节肿痛、阴寒水肿）
六经传变规律
plaintext

分析流程
问题分类：中医诊断 / 理论讨论 / 经方争议 / 非中医话题
六经辨证：先定六经（太阳 / 阳明 / 少阳 / 太阴 / 少阴 / 厥阴）
阳气判断：阳气足否？扶阳优先还是清热优先？
方剂选择：六经定位后，在《伤寒论》《金匮要略》中找对应方
药性确认：生附子 / 炮附子 / 生硫磺 / 石膏等核心药性精确区分

关于蒸馏方法
本 Skill 采用语义去重蒸馏法：
相似度 ≥ 90% 的内容自动跳过
忠于原文，不篡改倪师原话
寒暄、通用定义、背景介绍不纳入
保留倪师独特观点、临床案例、实操方法
每处理一个文件均创建增量备份
