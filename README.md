## 核心观点速览

### 生附子 vs 炮附子 vs 生硫磺

`
生附子 ──→ 阴寒（病在里、阳气极虚、四逆证：脚冷至膝/手冷至肘）
炮附子 ──→ 表虚（表阳不固、汗出恶风、卫气不固）
生硫磺 ──→ 水中火（命门火衰、水肿、关节肿痛、阴寒水肿）
`

### 六经传变规律

`
太阳（表）→ 阳明（里热）→ 少阳（半表半里）
    ↓ 失治误治
太阴（脾寒）→ 少阴（心肾阳虚）→ 厥阴（阴阳逆乱/上热下寒）
`

### 五味饮食观（倪师临床诊断延伸）

| 厚味（深色/浓味） | 伤害 | 淡味（浅色/清淡） | 归属 |
|---|---|---|---|
| 咖啡（浓酸） | 伤脾→伤肾→骨伤 | 蔬菜淡苦 | 入心 |
| 牛奶（酸性） | 伤脾/胰 | 蔗糖（天然甘） | 缓急止痛 |
| 重盐 | 伤骨/心 | 天然海盐 | 适量护正 |

## 使用方法

### 在 OpenClaw 中激活

在对话中直接使用激活词，即可切换到倪海厦视角分析问题：

`
你从倪海厦的角度分析一下：病人肺癌晚期，咳嗽有血，应该怎么处理？
`

### 触发词（任一即可）

`
倪海厦 / 海厦视角 / 倪师 / 经方思维 / 倪海厦会怎么看
用倪海厦的方式分析 / 从倪师的角度
`

### 分析流程

1. **问题分类**：中医诊断 / 理论讨论 / 经方争议 / 非中医话题
2. **六经辨证**：先定六经（太阳/阳明/少阳/太阴/少阴/厥阴）
3. **阳气判断**：阳气足否？扶阳优先还是清热优先？
4. **方剂选择**：六经定位后，在《伤寒论》《金匮要略》中找对应方
5. **药性确认**：生附子/炮附子/生硫磺/石膏等核心药性精确区分

## 关于蒸馏方法

本 Skill 采用**语义去重蒸馏法**：
- 相似度 ≥ 90% 的内容自动跳过
- 忠于原文，不篡改倪师原话
- 寒暄、通用定义、背景介绍不纳入
- 保留倪师独特观点、临床案例、实操方法
- 每处理一个文件均创建增量备份
